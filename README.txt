Repository of the code generating the results of the paper:
"A hybrid systems framework for data-based adaptive control of linear time-varying systems"
by Andrea Iannelli, Romain Postoyan.

The paper is available at: https://arxiv.org/abs/2405.14426

* The main file is "master", where high-level choices (plant & type of controller analyzed) are made; this routine in turn calls "HS_DAC", where all the other functions are called
* Results were generated using MATLAB2024a
* MOSEK and Yalmip required to run the SDPs.
